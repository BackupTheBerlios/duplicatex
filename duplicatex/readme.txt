Duplicatex v1.0:
================

Zweck:
Suchen, finden und markieren von identischen Files (Duplikaten)

Kompilieren & Installieren:
===========================

Linux:
------
Zum Kompilieren wird nur eine wxWidgets-Library (empfehlenswerter Weise die Version 2.4.2 - andernfalls
Makefile anpassen) benötigt: www.wxwidgets.org

Das Programm <duplicatex> kann von jeder beliebigen Stelle aus gestartet werden, zu empfehlen ist aber 
ein eigenes Unterverzeichnis.

Windows:
--------

duplicatex.msp: 
Projectfile fuer das Mingw-Developerstudio, alternativ laesst sich auch das linux-Makefile 
sehr leicht anpassen



Zum Kompilieren wird ebenfalls nur eine wxWidgets-Library benötigt. Für Windows empfehle ich
das Mingw-Developerstudio von:

http://petra.hos.u-szeged.hu/~aking/www.parinya.ca/www.parinyasoft.com/download.html

(Hier ist alles schon installationsfertig zusammengestellt, was benötigt wird. Inklusive wxWidgets)

Das Programm <duplicatex> kann von jeder beliebigen Stelle aus gestartet werden, zu empfehlen ist aber 
ein eigenes Unterverzeichnis.
